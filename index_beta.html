This tool trims a source tree set to a subset of species and generates random trees from that. Copy and paste species from this list (or the CSV file) into the box to the right, fill in the fields below, and then click "Get Trees" to download a zipped set of random trees with metadata.


A global phylogeny of birds

This webpage offers Phylogenetic Tree Sets from W. Jetz, Thomas, G. H. , Joy, J.B., Hartmann, K. and A.O. Mooers (in press): "The global diversity of birds in space and time". Nature.


Full sets of bird trees in flat newick format for both the Hackett and Ericson backbones, and for both DNA-data only (Stage 1, 6670 species) trees and complete (Stage 2, 9993 species) trees. Each file contains 1,000 trees from the posterior distribution.


Complete taxonomy that these trees were based on (CSV).
